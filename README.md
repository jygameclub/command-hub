# Command Hub

一个纯前端的命令/笔记/URL 管理工具，使用 SQLite (sql.js) 在浏览器本地存储数据。

## 功能特性

- **多标签管理** - 支持创建多个 Tab，每个 Tab 可设置为命令、笔记或 URL 类型
- **All 标签** - 系统内置的全局视图，显示所有内容并按复制次数排序
- **命令类型** - 每行一个命令，一键复制到剪贴板，自动统计复制次数
- **URL 类型** - 每行一个链接，一键在新标签页打开，右侧边栏快捷访问
- **笔记类型** - 自由文本内容存储
- **数据导入导出** - 支持 .db 数据库文件和 JSON 格式导入导出
- **主题切换** - 多种深色/浅色主题可选，支持自定义颜色
- **本地存储** - 数据保存在浏览器 IndexedDB，无需后端服务器

## 快速开始

1. 直接用浏览器打开 `index.html` 文件
2. 或使用任意静态文件服务器：
   ```bash
   # 使用 Python
   python -m http.server 8080

   # 使用 Node.js
   npx serve
   ```
3. 访问 http://localhost:8080

## 技术栈

- 原生 HTML/CSS/JavaScript
- [sql.js](https://github.com/sql-js/sql.js) - 浏览器端 SQLite 实现
- IndexedDB - 数据持久化存储

## 使用说明

### 标签管理
- 点击 **+ 新建 Tab** 创建新标签
- 点击 **编辑当前 Tab** 修改标签名称、类型或删除
- 拖拽或使用左右移动按钮调整标签顺序

### 项目管理
- 点击 **+ 添加项目** 在当前标签添加内容
- 命令/URL 类型支持多行，每行作为独立项
- 点击复制/打开按钮，系统自动统计使用次数

### 数据管理
- **导入/导出** - 支持整个数据库 (.db) 的导入导出
- **JSON 导入导出** - 针对单个 Tab 的 JSON 格式导入导出
- **加载默认数据库** - 在设置中可加载预设的示例数据

### 设置
- 主题切换（深色/浅色多种主题）
- 自定义命令和描述文字颜色
- 字体大小调节
- 显示/隐藏 All 标签
